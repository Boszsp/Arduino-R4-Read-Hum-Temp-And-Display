# Arduino-R4-Read-Hum-Temp-And-Display
Arduino R4 Read Hum Temp And Display With Adafruit Lib
## Connection With Sensor Guide
+ Pattern
  + `Sensor` -> `Arduino`
+ LCD Display 16x2 I2C
  + `GND` -> `GND`
  + `VCC` -> `5v`
  + `SDA` -> `A4`
  + `SCL` -> `A5`
+ SHT31 Hum&Temp Sensor
  + `GND` -> `GND`
  + `VCC` -> `3.3v`
  + `SCK` -> `SCL`
  + `DATA` -> `SDA`
